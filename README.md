# Memo.md (NEMO)

바탕화면 메모 위젯 + 옵시디언(MD) 내보내기. 현재 버전: 1.8.3

## 이 앱을 만든 이유

바탕화면에 쓰는 메모는 아무렇게나 흩어지기 쉬워서, 나중에 찾거나 서로 연결하기가 어렵습니다. 처음부터 주제별로 나눠서 쓰면 나중에 기억하고 연결하기가 훨씬 쉬워지지 않을까 생각했습니다.

옵시디언(Obsidian)을 써보고 싶었지만 마크다운 문법이 낯설어서 시작부터 막막했습니다. 다시 들여다보니 카테고리 분류와 메모 연결만 잘 되면 마크다운을 몰라도 문제없겠다 싶어서, 혼자 쓰려고 이 앱을 만들었습니다.

"메모 하나 쓰는데 무슨 주제를 정해" 라고 할 수도 있지만, 사실 처음 딱 한 번만 카테고리/주제를 설정해두면 그다음부터는 다른 메모앱과 다를 게 없습니다. 일단 아무렇게나 쓰고 나중에 다른 주제로 옮길 수도 있고요. 위젯의 주제 버튼 한 번 클릭하거나 단축키만 누르면 바로 새 메모가 열립니다. 목표는 작은 메모 하나도 놓치지 않고 저장하면서, 동시에 나중에 옵시디언의 카테고리 구조에 맞게 정리돼 있는 것이었습니다.

마크다운을 잘 모르는 저 같은 사람도 옵시디언의 분류 체계에 맞게 편하게 메모를 쌓을 수 있게 하는 게 이 앱의 목적입니다.

*이 앱은 코딩을 모르는 비개발자가 Claude(AI)의 도움을 받아 만든 것이라, 부족하거나 어설픈 부분이 있을 수 있습니다.*




## 1. 실행 준비 (최초 1회만)

1. Node.js 설치 (없다면): https://nodejs.org 에서 LTS 버전 설치
2. 이 폴더에서 터미널(명령 프롬프트) 열기
3. 아래 명령어 입력 후 엔터 (필요한 부품 설치, 몇 분 걸림)
   ```
   npm install
   ```

## 2. 테스트 실행

```
npm start
```
- 실행하면 위젯이 뜨고, 트레이(우측 하단 시계 옆)에 아이콘이 생깁니다.
- 트레이 아이콘: 클릭 = 위젯 열기/앞으로, 더블클릭 = 새 메모, 우클릭 = 메뉴(설정/도움말/종료)

## 3. 진짜 실행파일(.exe)로 만들기

테스트해보고 마음에 들면, **실행 중인 앱을 먼저 종료한 뒤** 아래 명령으로 설치 파일을 만듭니다.

```
npm run build-win
```
- 완료되면 `dist` 폴더 안에 설치 파일(.exe)이 생성됩니다.
- 기존 버전 위에 그대로 덮어 설치해도 됩니다 — 메모/설정 데이터는 별도 위치(AppData)에 있어서 유지됩니다.

## 4. 처음 사용할 때 꼭 할 것

1. 설정 > 주제관리에서 **주제 먼저 등록** (주제가 있어야 메모를 만들 수 있어요)
2. MD(마크다운) 기능을 쓸 거면 설정 > MD설정에서 **Vault(저장) 폴더 지정**, 안 쓸 거면 MD기능 체크 해제
3. 위젯의 주제 버튼을 눌러 메모 생성 테스트

## 5. 주요 기능 요약 (자세한 건 앱 안의 도움말 참고)

- **자동 저장**: 입력하는 동안 0.5초 단위로 자동 저장. 저장 파일은 안전장치(임시파일 교체 + 직전본 백업 + 자동복구)로 보호됩니다.
- **위젯**: 주제 버튼으로 새 메모, 목록에서 메모 열기/숨기기, 드래그로 순서 변경, 타이틀바 더블클릭으로 접기. 접힘 상태에서 주제 버튼 더블클릭 = 그 주제 숨김/보임.
- **눈(👁) 버튼**: 누를 때마다 "다 숨김 → 직전 상태로 → 다 보임 → 직전 상태로" 순환. 숨김 상태는 재부팅해도 기억됩니다.
- **🔍 검색**: 모든 메모의 제목+본문을 실시간 검색, 클릭하면 그 메모가 열림.
- **휴지통**: 지운 메모는 설정 > 휴지통에 60일 보관(첨부 포함), 복구/완전삭제/비우기 가능.
- **MD내보내기**: 메모창 하단 버튼으로 Vault 폴더에 .md 저장(첨부는 "첨부" 폴더로 자동 복사). 내보낸 후 유지/자동삭제는 메모별로 지정 가능.
- **서식 도구(메모창 하단)**: 굵게/기울임/취소선/밑줄/형광펜/코드/링크/첨자/목록/제목/들여쓰기 — 전부 마크다운 문법으로 삽입됩니다. MD기능을 끄면 서식 줄은 숨겨집니다.
- **체크리스트**: 본문 줄들을 체크박스 목록으로 변환, 내보낼 때 `- [ ]` 문법으로 포함.
- **템플릿**: 메모의 본문+이미지+체크리스트+창 크기를 주제 템플릿으로 저장, 새 메모에 자동 적용.
- **백업**: 설정 > 일반에서 전체 내보내기(수동)와 자동 백업(주기 지정) 지원. 백업 폴더에서 복구도 가능.
- **전역 단축키**: 기본 Ctrl+Alt+N으로 어디서든 새 메모 (설정에서 변경/해제 가능).
- **기타**: 메모별 색상/항상위/접기, 이미지 자유 배치·캡션, 메모 연결([[링크]]), 투명도 조절, 이중 실행 방지.

간단 기능 소개

-주제 설정을 해야 메모를 작성할수 있어요(한번만 수정및 편집가능)-(위젯)
설정해놓은 주제의 버튼이 생성 되는데 버튼을 클릭하거나 단축키로 새 메모가 생성 돼요
주제버튼의 색상과 메모지 색상으로 구분및 주제별 컨트롤 가능(항상위,숨김)

-위젯의 메모대쉬보드를 접었다 펼수있어요(위젯)
대시보드엔 주제별 개별 메모 요약 주제멸 목록펼침및 접기(토글)
대시보드 주제순서 바꾸기(버튼의 순서도 같이 바뀌어요)
위젯이 접힌 상태에서는 버튼으로만 컨트롤

-주제숨김(위젯)
남들에게 보이기 싫은 주제는 숨김으로 설정할수 있어요
위젯에서 우측 바를 클릭하면 숨김주제가 토글 돼요

-검색 기능(위젯)
작성한 메모를 검색할수 있어요

-메모링크(메모지)
메모를 md로 내보낸 파일에 링크 버튼으로 링크할수 있어요
링크버튼을 누르면 내보낸 md파일 목록이 불러와져요.검색으로 찾을수있어요

-체크리스트(메모지)
메모전체를 체크리스트로 변경하거나 본문으로 돌릴수있어요
지정한 문구만 체크리스트 변환가능

-이미지 삽입/첨부파일(메모지)
드래그로 이미지 삽입및 파일 첨부 가능
이미지는 큰 이미지도 1500px로 변환돼요
이미지에 설명 티커가 따라붙어요

-탬플릿 설정(메모지)
 메모의 내용을 템플릿으로 정할수 있어요
 템플릿 지정 주제는 새 메모를 만들면 항상 템플릿이 반영된 메조지가 생성돼요

-해당 메모를 Md파일로 지정된 폴더에 저장해요(메모지)
저장된 메모는 삭제되거나 그냥 두거나 설정창 기본값을 따를수도 있고 개별적 선택이 가능해요(메모지 동그란 버튼)

-편집바에 자주사용하는 특수문자를 등록해서 사용할수 있어요(설정)

-저장폴더(vault) 지정 및 백업폴더 지정(설정)
저장 폴더및 백업폴더를 지정해서옵시디언에서 바탕화면 메모를 따로 관리 할수있어요

-휴지통 기능(설정)
삭제한 메모는 휴지통에 60일간 보관됩니다

-자동백업(설정)
6시간 12시간 매일 간격으로 자동백업 설정할수 있어요

-파일 내보내기(설정)
백업이나 저장폴더와는 별개로
txt나md파일로 전부 내보낼수 있어요
