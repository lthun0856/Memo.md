# Memo.md (NEMO)

바탕화면 메모 위젯 + 옵시디언(MD) 내보내기. 현재 버전: 1.8.3

## 1. 실행 준비 (최초 1회만)

1. Node.js 설치 (없다면): https://nodejs.org 에서 LTS 버전 설치
2. 이 폴더에서 터미널(명령 프롬프트) 열기
3. 아래 명령어 입력 후 엔터 (필요한 부품 설치, 몇 분 걸림)
   ```
   npm install
   ```

## 2. 테스트 실행

```
npm start
```
- 실행하면 위젯이 뜨고, 트레이(우측 하단 시계 옆)에 아이콘이 생깁니다.
- 트레이 아이콘: 클릭 = 위젯 열기/앞으로, 더블클릭 = 새 메모, 우클릭 = 메뉴(설정/도움말/종료)

## 3. 진짜 실행파일(.exe)로 만들기

테스트해보고 마음에 들면, **실행 중인 앱을 먼저 종료한 뒤** 아래 명령으로 설치 파일을 만듭니다.

```
npm run build-win
```
- 완료되면 `dist` 폴더 안에 설치 파일(.exe)이 생성됩니다.
- 기존 버전 위에 그대로 덮어 설치해도 됩니다 — 메모/설정 데이터는 별도 위치(AppData)에 있어서 유지됩니다.

## 4. 처음 사용할 때 꼭 할 것

1. 설정 > 주제관리에서 **주제 먼저 등록** (주제가 있어야 메모를 만들 수 있어요)
2. MD(마크다운) 기능을 쓸 거면 설정 > MD설정에서 **Vault(저장) 폴더 지정**, 안 쓸 거면 MD기능 체크 해제
3. 위젯의 주제 버튼을 눌러 메모 생성 테스트

## 5. 주요 기능 요약 (자세한 건 앱 안의 도움말 참고)

- **자동 저장**: 입력하는 동안 0.5초 단위로 자동 저장. 저장 파일은 안전장치(임시파일 교체 + 직전본 백업 + 자동복구)로 보호됩니다.
- **위젯**: 주제 버튼으로 새 메모, 목록에서 메모 열기/숨기기, 드래그로 순서 변경, 타이틀바 더블클릭으로 접기. 접힘 상태에서 주제 버튼 더블클릭 = 그 주제 숨김/보임.
- **눈(👁) 버튼**: 누를 때마다 "다 숨김 → 직전 상태로 → 다 보임 → 직전 상태로" 순환. 숨김 상태는 재부팅해도 기억됩니다.
- **🔍 검색**: 모든 메모의 제목+본문을 실시간 검색, 클릭하면 그 메모가 열림.
- **휴지통**: 지운 메모는 설정 > 휴지통에 60일 보관(첨부 포함), 복구/완전삭제/비우기 가능.
- **MD내보내기**: 메모창 하단 버튼으로 Vault 폴더에 .md 저장(첨부는 "첨부" 폴더로 자동 복사). 내보낸 후 유지/자동삭제는 메모별로 지정 가능.
- **서식 도구(메모창 하단)**: 굵게/기울임/취소선/밑줄/형광펜/코드/링크/첨자/목록/제목/들여쓰기 — 전부 마크다운 문법으로 삽입됩니다. MD기능을 끄면 서식 줄은 숨겨집니다.
- **체크리스트**: 본문 줄들을 체크박스 목록으로 변환, 내보낼 때 `- [ ]` 문법으로 포함.
- **템플릿**: 메모의 본문+이미지+체크리스트+창 크기를 주제 템플릿으로 저장, 새 메모에 자동 적용.
- **백업**: 설정 > 일반에서 전체 내보내기(수동)와 자동 백업(주기 지정) 지원. 백업 폴더에서 복구도 가능.
- **전역 단축키**: 기본 Ctrl+Alt+N으로 어디서든 새 메모 (설정에서 변경/해제 가능).
- **기타**: 메모별 색상/항상위/접기, 이미지 자유 배치·캡션, 메모 연결([[링크]]), 투명도 조절, 이중 실행 방지.
