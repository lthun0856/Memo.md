Memo.md 사용 설명서

바탕화면에 상주하는 메모장이에요. 메모를 주제별로 정리하고, MD(마크다운) 파일로 내보낼 수 있어요. 옵시디언 없이도 쓸 수 있고, 옵시디언 Vault 폴더를 저장 위치로 지정하면 옵시디언과도 바로 연동돼요.

🖱트레이 아이콘

작업표시줄 오른쪽 아래(숨겨진 아이콘 포함) 트레이 아이콘을 우클릭하면 메뉴가 나와요.

  • 일반 새 메모 / 주제별 새 메모 — 새 메모창을 엽니다.
  • 위젯 열기/닫기 — 위젯을 껐다 켭니다.
  • 설정 — 설정 화면을 엽니다.
  • 도움말 — 지금 보고 있는 이 화면을 엽니다.
  • 환영 화면 다시 보기 — 처음 실행할 때 나왔던 소개 화면을 다시 봅니다.
  • 종료 — 앱을 완전히 끕니다.

트레이 아이콘을 그냥 클릭(좌클릭)하면 위젯이 열리거나 앞으로 나와요. 더블클릭하면 새 메모가 하나 생겨요.

🗂위젯

화면에 항상 떠있는 작은 창으로, 주제 아이콘과 메모 목록을 보여줘요.

먼저 주제를 작성해야 메모를 할 수 있어요. 이 앱은 메모를 주제별로 분류해서 관리하는 게 핵심이에요. 설정 > 주제관리에서 주제를 먼저 만들고, 그 주제에 맞게 메모를 작성해보세요. 주제별로 색상·텍스트를 정할 수 있고, 위젯에 생기는 주제 버튼으로 바로바로 메모할 수 있어서 메모 관리가 훨씬 수월해져요.

  • 주제 아이콘 / 주제 칸 — 클릭·더블클릭·Ctrl+클릭에 따라 새 메모 만들기, 숨기기·보이기, 앞으로 가져오기 등 여러 동작을 해요. 클릭 방식별로 정확히 뭘 하는지는 바로 다음 위젯 토글 섹션에 전부 모아뒀어요.
  • 목록의 × 버튼 — 그 메모를 바로 삭제합니다.
  • 👁 버튼(주제별) · 핀 모양 버튼(주제별) · 위젯 타이틀바의 👁·핀·더블클릭 — 전부 상태를 켜고 끄는 토글 동작이라 위젯 토글 섹션에서 한 번에 설명해요.
  • 위젯 타이틀바의 🔍 — 작은 검색창이 뜹니다. 제목과 본문 내용에서 찾아서 주제별로 묶어 보여주고, 목록에서 클릭하면 그 메모가 바로 열려요.

위젯 세로 크기는 마우스로 직접 늘리거나 줄일 수 없고, 목록을 펼치고 접는 상황과 메모 개수에 맞춰 항상 자동으로 정확하게 맞춰져요(가로 크기는 드래그로 조절 가능). 위젯의 마지막 위치는 다음에 앱을 켤 때도 그대로 기억됩니다.

🔁위젯 토글 동작 모음

위젯에는 누를 때마다 상태가 바뀌는 "토글" 동작이 많아서, 헷갈리지 않게 여기 한 곳에 전부 모아뒀어요.

주제 아이콘 — 위젯을 펼쳤을 때 (상단 아이콘 줄)

  • 클릭 — 그 주제로 새 메모를 만듭니다.

주제 아이콘 — 위젯을 접었을 때

  • 한 번 클릭 — 새 메모를 만듭니다.
  • 두 번 클릭(더블클릭) — 그 주제 메모 전체를 숨기기 ↔ 보이기로 토글합니다.

주제 칸 — 위젯을 펼쳤을 때 (대시보드 목록)

  • 한 번 / 두 번 클릭 — 설정 > 위젯 탭에서 정한 동작이 실행돼요. 기본은 한 번 클릭 = 목록 펼치기 ↔ 접기, 두 번 클릭 = 그 주제 메모 전체 열기 ↔ 숨기기 토글이고, "새 메모 생성"이나 "아무 동작 안 함"으로 바꿀 수도 있어요. 목록은 주제마다 따로 펼쳐지고 접혀서 여러 주제를 동시에 펼쳐놓을 수 있어요.

Ctrl+클릭 — 주제 아이콘·주제 칸 어디서든 동일 (토글 아님)

  • 그 주제의 지금 화면에 보이는(숨겨지지 않은) 메모창들을 다른 프로그램 창보다 맨 앞으로 가져옵니다. 누를 때마다 항상 앞으로만 가져오고, 뒤로 보내는 동작은 따로 없어요. 메모창은 평소 다른 작업을 방해하지 않으려고 조용히 뜨기 때문에 다른 프로그램 창이 앞에 있으면 가려질 수 있는데, 필요할 때 이걸로 꺼내볼 수 있어요.

눈(👁) 아이콘

  • 주제별 눈 아이콘(대시보드 목록) — 그 주제 메모를 숨기기 ↔ 보이기로 토글합니다.
  • 위젯 타이틀바의 눈 아이콘 — 누를 때마다 ①모든 메모 숨기기 → ②누르기 전 상태로 복원 → ③모든 메모 보이기 → ④누르기 전 상태로 복원 순서로 4단계 순환합니다. 일부러 숨겨둔 주제는 ②·④에서도 그대로 숨겨진 채 돌아와요. 이 상태는 컴퓨터를 껐다 켜도 기억돼요.

핀 모양 아이콘 (항상 위에 표시)

  • 주제별 핀 아이콘 — 그 주제 메모들을 항상 위에 표시할지 켜기 ↔ 끄기로 토글합니다.
  • 위젯 타이틀바의 핀 아이콘 — 위젯 자체를 항상 위에 표시할지 켜기 ↔ 끄기로 토글합니다.

기타

  • 위젯 타이틀바 더블클릭 — 위젯 자체를 접기 ↔ 펼치기로 토글합니다. 접으면 주제 아이콘만 남아요.
  • 목록에서 메모 제목 클릭 — 안 열려있으면 열고, 열려있으면 닫는 것으로 토글합니다(위 주제 칸 클릭 설정과는 무관하게 항상 이렇게 동작해요).

📝메모창

  • 주제칸(맨 왼쪽) 더블클릭 — 메모 제목을 입력할 수 있어요.
  • ＋ — 같은 주제로 메모를 하나 더 만듭니다.
  • 색상 아이콘 — 이 메모의 색을 바꿉니다.
  • (핀 모양) — 이 메모만 항상 위에 표시합니다.
  • 기본값 버튼 — MD내보내기 후 이 메모를 바탕화면에 남길지, 자동으로 지울지 정합니다(설정의 기본값을 따르거나, 반대로 동작하게 바꿀 수 있어요).
  • 🗑 — 메모를 삭제해서 휴지통으로 보냅니다(설정 > 휴지통에서 60일간 복구 가능).
  • ▾ (접기 버튼, 🗑와 × 사이에 위치) — 메모창을 접었다 폅니다. 접으면 제목줄만 남아요.
  • × — 메모창을 닫습니다(삭제되는 건 아니에요, 위젯에서 다시 열 수 있어요).

메모창은 편집 중이 아닐 때는 대부분의 버튼이 자동으로 숨겨져서 깔끔하게 보여요(접기 버튼과 닫기 버튼은 항상 보여요). 메모창에 마우스를 올리거나 클릭하면 나머지 버튼이 다시 나타납니다. 또한 위젯을 포함한 모든 창은 작업표시줄에는 나타나지 않고, 트레이 아이콘으로만 관리돼요.

서식 도구 (메모창 아래쪽)

메모 본문은 서식 있는 글이 아니라 마크다운(MD) 텍스트예요. 서식 버튼을 누르면 글자가 화면에서 바로 굵어지거나 색이 바뀌는 게 아니라, 선택한 글자 앞뒤에 마크다운 기호(**, * 등)가 붙어요. 그 상태로 MD내보내기해서 옵시디언 같은 마크다운 뷰어로 열면 그때 굵게·기울임 등으로 보여요.

  • 실행취소/다시실행, 글머리 목록(- ), 들여쓰기, 제목 크기(#~### 등)
  • 굵게 — 선택한 글자 앞뒤에 **를 붙여요. 이미 **로 감싸진 글자를 다시 선택해서 누르면 기호가 제거돼요(토글).
  • 기울임(*), 취소선(~~), 밑줄(<u>), 형광펜(==), 코드(` 또는 여러 줄이면 ```), 위/아래 첨자(<sup>/<sub>) — 전부 같은 방식으로 선택한 글자를 해당 마크다운(또는 HTML) 기호로 감싸요.
  • 링크 — 선택한 글자에 인터넷 주소를 연결합니다.
  • 파일/이미지 첨부 — 파일을 골라 첨부하거나, 이미지를 그림판처럼 메모 위에 자유롭게 놓을 수 있어요(드래그로 이동, 모서리를 끌어서 크기 조절). 이미지만은 예외적으로 화면에 실제 그림으로 보여요.
  • 특수문자 버튼 — 설정 > "편집바 특수문자"에서 등록해둔 문자가 툴바 맨 끝에 버튼으로 나와요. 누르면 커서 위치에 그 글자가 바로 써져요.

이미지는 화면에는 그림으로만 보이고, MD내보내기할 때 자동으로 파일에 포함돼요.

하단 버튼

  • 복사 — 메모 전체 내용을 클립보드에 복사합니다.
  • 불러오기(⤒) — txt/md 파일을 불러와 메모에 넣습니다.
  • txt로 저장 — 메모 내용을 텍스트 파일로 저장합니다.
  • 메모 연결 — 다른 메모로 가는 링크를 겁니다. (아래 메모 연결 참고)
  • MD내보내기 — 메모를 MD(마크다운) 파일로 내보냅니다. (아래 참고)

☑️체크리스트

메모 본문은 순수 텍스트라서 그 안에서는 진짜 체크박스나 취소선을 그릴 수 없어요. 그래서 체크리스트는 이미지 첨부처럼 본문과는 별도인 목록으로 관리돼요.

  • 체크리스트 만들기 — 본문에 할 일을 한 줄씩 적은 뒤, 툴바의 체크리스트 버튼(☑)을 누르세요. 글자를 선택한 상태면 선택한 줄들만, 아무것도 선택 안 했으면 메모 전체가 체크박스 목록으로 바뀌고 본문에서는 그 줄들이 사라져요.
  • 체크 표시 — 체크박스를 누르면 그 줄 텍스트에 실제로 취소선이 그어져서 완료 표시가 돼요. 다시 누르면 취소선이 없어지고 원래대로 돌아가요(몇 번을 껐다 켜도 내용은 그대로 유지돼요).
  • 내용 수정 — 항목의 글자를 클릭하면 바로 그 자리에서 고칠 수 있어요.
  • 항목 삭제 — 항목 오른쪽의 × 버튼으로 그 항목만 지울 수 있어요.
  • 항목 추가 — 목록 맨 아래 "+ 항목 추가"를 누르면 새 항목이 바로 생기고 입력 상태가 돼요. 체크리스트를 만들었다 풀었다 할 필요 없이 계속 이 상태로 추가하면 돼요.

MD내보내기·txt로 저장·전체 복사를 할 때 체크리스트는 본문 뒤에 - [ ] 할 일 / - [x] 완료한 일 형식의 마크다운으로 자동으로 붙어요. 옵시디언은 이 문법을 그대로 체크박스로 인식해요. 다만 내보낸 뒤 옵시디언에서 체크박스를 바꿔도 앱으로 다시 반영되진 않아요(내보내는 순간에만 반영되는 한 방향이에요).

🔗메모 연결

다른 메모로 바로 찾아갈 수 있게, 메모 안에 다른 메모로 가는 링크([[파일명]])를 걸어주는 기능이에요. 서식 도구의 "링크"(인터넷 주소 연결)와는 별개예요.

  • 메모 연결 버튼(메모창 하단) — 누르면 별도의 작은 검색 팝업창이 떠요.
  • 팝업에서 검색·선택 — 제목이나 주제로 검색해서 원하는 메모를 고르면, 커서 위치에 [[파일명]] 형식으로 바로 삽입돼요.

목록에는 이미 옵시디언으로 MD내보내기한 메모만 나와요. 아직 내보내지 않은 메모는 정확한 파일명을 알 수 없어서 목록에 뜨지 않아요. 옵시디언은 [[파일명]] 문법을 그 메모 파일로 가는 링크로 자동 인식해요. 이 기능은 설정 > MD설정에서 "MD 기능 사용"이 켜져 있을 때만 쓸 수 있어요.

📤MD내보내기

이 앱은 원래 옵시디언과 함께 쓰려고 만들었지만, 그냥 MD(마크다운) 파일로 저장하는 기능이라 옵시디언 없이도 쓸 수 있어요. 옵시디언을 쓴다면 저장 폴더를 옵시디언 Vault 폴더로 지정하기만 하면 자동으로 연동돼요.

MD내보내기 버튼을 누르면 저장할 파일명을 미리 채워서 보여줘요. 그대로 저장하거나 수정할 수 있어요.

기본 파일명 규칙: 주제_제목_YYYYMMDD_001

  • 주제가 없으면 "미분류", 제목이 없으면 "제목없음"으로 들어가요.
  • 번호(001, 002…)는 같은 주제·제목·날짜로 이미 저장된 파일이 있으면 다음 번호가 되고, 날짜가 바뀌면 001부터 다시 시작돼요.
  • 주제가 있는 메모는 저장 폴더 안에 그 주제 이름 폴더로, 없는 메모는 저장 폴더 최상위에 저장돼요.
  • 저장 전에는 반드시 설정 > MD내보내기 탭에서 저장 폴더를 먼저 지정해야 해요.

파일명 규칙 직접 조정 — 설정 > MD내보내기 탭의 "MD내보내기 파일명 규칙"에서 주제/제목/날짜/번호 항목을 각각 켜고 끌 수 있고, 구분자(밑줄 _ / 하이픈 - / 공백)도 고를 수 있어요. 순서는 항상 주제 → 제목 → 날짜 → 번호로 고정이에요. 설정 화면에 바로 예시가 보여서 미리 확인할 수 있어요.

  • [주제관리] 먼저 주제를 작성해야 메모를 할 수 있어요. 주제 추가/수정/삭제, 아이콘·색상(아이콘배경/아이콘글자/메모배경) 지정, 짧은 설명, 새 메모에 자동으로 채워질 기본 제목, 위젯에서 숨김 여부를 정할 수 있어요. 카테고리(상위주제)는 주제를 묶어서 보여주는 이름표로, MD 기능이 켜져 있을 때 주제 이름에 /를 넣으면 옵시디언에서 중첩 폴더·태그로도 정리돼요. 주제의 메모배경 색을 바꾸면 그 색을 그대로 쓰던 기존 메모들도 색이 같이 바뀌어요(메모창에서 직접 다른 색으로 바꿔둔 메모는 그대로 유지돼요).
  • [일반] 윈도우 시작 시 자동 실행, 메모 여러 개 동시에 띄우기, 메모 삭제할 때 확인창 표시, 화면 투명도(10%~100%, 위젯+모든 메모창 공통)를 정합니다. 단축키를 등록하면 다른 프로그램을 쓰는 중에도 바로 새 메모를 만들 수 있어요(직전에 작업하던 주제에 생성돼요). 전체 메모 내보내기로 모든 메모를 한꺼번에 txt/md로 저장하거나, 백업에서 복구하기로 백업 폴더의 메모를 다시 불러올 수 있어요(항상 추가만 되고 기존 메모는 안 지워짐). 저장공간 정리는 어떤 메모에서도 안 쓰는 첨부파일만 찾아 지워줘요. 자동 백업을 켜두면 지정한 주기마다 md로 자동 저장되는데, 항상 같은 파일에 덮어써서 용량이 계속 쌓이지 않아요.
  • [위젯] 자동 크기조절, 가로 크기, 클릭 동작, 상단바 색상을 정합니다(세로 크기는 항상 자동). 클릭 동작(한 번/두 번 클릭)은 주제칸을 클릭할 때만 적용되고, 목록 안 개별 메모 클릭은 이 설정과 상관없이 항상 열기/닫기로 동작해요. 클릭 동작에 대한 자세한 설명은 위젯 토글 섹션 참고.
  • [MD설정] MD 기능 사용 스위치로 마크다운/옵시디언 관련 기능 전체를 켜고 끌 수 있어요 — 끄면 메모창의 서식 도구, 메모 연결, MD내보내기 버튼, 처리방식 버튼, 주제관리의 "상위 주제" 선택이 전부 숨겨지고 순수한 메모장으로만 동작해요. 그 외에 저장 폴더(Vault) 지정, MD내보내기 후 동작(기본값), 내보내기 파일명에 주제/제목/날짜/번호를 포함할지·구분자는 뭘 쓸지도 여기서 정합니다.
  • [편집바 특수문자] 메모창 서식 툴바에 바로 눌러 넣을 수 있는 특수문자 버튼을 만들 수 있어요. 사용하고 싶은 특수문자를 직접 넣어주세요. 몇 개를 넣을지 정하고, 각 칸에 원하는 문자를 입력하면 메모창 서식 툴바에 버튼으로 나타나요.
  • [휴지통] 삭제한 메모가 60일간 보관돼요. 목록에서 복구하거나 완전삭제할 수 있고, 휴지통 비우기로 한 번에 다 지울 수도 있어요. 60일이 지난 항목은 자동으로 완전히 지워져요.
  • [도움말] 지금 보고 있는 이 설명서를 설정창 안에서 바로 볼 수 있어요.
